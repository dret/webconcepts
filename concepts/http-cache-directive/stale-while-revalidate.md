[When present in an HTTP response, the stale-while-revalidate Cache-Control extension indicates that caches MAY serve the response in which it appears after it becomes stale, up to the indicated number of seconds.](http://tools.ietf.org/html/rfc5861#section-3 "Read documentation for HTTP Cache Directive &#34;stale-while-revalidate&#34;") (**[RFC 5861: HTTP Cache-Control Extensions for Stale Content](/specs/IETF/RFC/5861 "This document defines two independent HTTP Cache-Control extensions that allow control over the use of stale responses by caches.")**)
