[This URI scheme provides information that can be used for sending SMS message(s) to specified recipient(s). The functionality is comparable to that of the "mailto" URI, which can also be used with a comma-separated list of email addresses.](https://datatracker.ietf.org/doc/html/rfc5724#section-2 "Read documentation for URI Scheme &#34;sms&#34;") (**[RFC 5724: URI Scheme for Global System for Mobile Communications (GSM) Short Message Service (SMS)](/specs/IETF/RFC/5724 "This memo specifies the Uniform Resource Identifier (URI) scheme &#34;sms&#34; for specifying one or more recipients for an SMS message. SMS messages are two-way paging messages that can be sent from and received by a mobile phone or a suitably equipped networked device.")**)
