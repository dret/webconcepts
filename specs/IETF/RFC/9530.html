Organization: Internet Engineering Task Force (IETF)
Series: Request for Comments (RFC)
Abstract: This document defines HTTP fields that support integrity digests. The Content-Digest field can be used for the integrity of HTTP message content. The Repr-Digest field can be used for the integrity of HTTP representations. Want-Content-Digest and Want-Repr-Digest can be used to indicate a sender's interest and preferences for receiving the respective Integrity fields.


Specified Web Concepts:

HTTP Header Fields

Content-Digest , Repr-Digest , Want-Content-Digest , Want-Repr-Digest
