Document Name: RFC 8142
Document URI: urn:ietf:rfc:8142
Online Version: https://datatracker.ietf.org/doc/html/rfc8142
Organization: Internet Engineering Task Force (IETF)
Series: Request for Comments (RFC)
Abstract: This document describes the GeoJSON text sequence format and "application/geo+json-seq" media type. This format is based on JavaScript Object Notation (JSON) Text Sequences and GeoJSON, and makes arbitrarily large geographic datasets incrementally parseable without restricting the form of GeoJSON texts within a sequence.


Specified Web Concepts:

Media Types
